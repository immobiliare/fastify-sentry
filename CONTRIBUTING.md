# Contributing

### npm scripts

```bash
# install dependencies
$ npm i
# run tests
$ npm test
# run the code linter
$ npm run lint
```

### Commit Convention

Angular.

### Test

Contributions should pass existing tests or have test cases for new functionality.

```bash
# Should pass
$ npm test
```

### Style

Style and lint errors should be fixed with

```bash
$ npm run lint
```
